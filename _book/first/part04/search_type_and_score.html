1. 搜索类型

1.1. query_then_fetch

GET /my_store/_search?search_type=query_then_fetch
{
  "query": {
    "match": {
      "productID": "KDKE"
    }
  }
}

1.2. dfs_query_then_fetch

GET /my_store/_search?search_type=dfs_query_then_fetch
{
  "query": {
    "match": {
      "productID": "KDKE"
    }
  }
}
